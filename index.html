Cara Instalasi Ekstensi

Pilih metode instalasi yang paling sesuai untuk Anda.

Instalasi Mudah (Unpacked Extension)

Metode ini memungkinkan Anda menginstal ekstensi secara langsung di browser Chrome Anda. Ini adalah metode yang direkomendasikan.

  Unduh Ekstensi WA Sender (ZIP)

Langkah-langkah Instalasi:

  1. Unduh File ZIP: Klik tombol "Unduh Ekstensi WA Sender (ZIP)" di atas untuk mendapatkan file WA_Sender_Free.zip.
  2. Ekstrak File: Setelah diunduh, ekstrak isi file WA_Sender_Free.zip ke folder baru di komputer Anda. Pastikan Anda mengingat lokasi folder ini.
  3. Buka Halaman Ekstensi Chrome: Buka browser Chrome Anda, ketik chrome://extensions/ di bilah alamat, lalu tekan Enter.
  4. Aktifkan Mode Pengembang: Di pojok kanan atas halaman Ekstensi, aktifkan tombol "Mode pengembang" (Developer mode).
  5. Muat Ekstensi: Klik tombol "Muat tanpa paket" (Load unpacked) yang muncul setelah Mode Pengembang diaktifkan.
  6. Pilih Folder Ekstensi: Pilih folder tempat Anda mengekstrak file WA_Sender_Free.zip pada Langkah 2.
  7. Selesai! Ekstensi WA Sender sekarang sudah terinstal dan siap digunakan. Anda akan melihat ikon WA Sender di bilah alat browser Anda.

Penting: Jangan hapus folder ekstensi yang telah diekstrak. Ekstensi memerlukan folder tersebut untuk tetap berfungsi.

Instalasi CRX (Packed Extension)

Metode ini menggunakan file .crx yang merupakan paket ekstensi siap instal. Ini mungkin sedikit kurang intuitif dibandingkan metode ZIP.

  Unduh Ekstensi WA Sender (CRX)

Langkah-langkah Instalasi:

  1. Unduh File CRX: Klik tombol "Unduh Ekstensi WA Sender (CRX)" di atas untuk mendapatkan file WA_Sender_Free.crx.
  2. Aktifkan Mode Pengembang: Di pojok kanan atas halaman Ekstensi, aktifkan tombol "Mode pengembang" (Developer mode).
  3. Seret dan Lepas File CRX: Seret file WA_Sender_Free.crx yang telah Anda unduh ke halaman chrome://extensions/.
  4. Konfirmasi Instalasi: Sebuah pop-up konfirmasi akan muncul. Klik "Tambahkan ekstensi" untuk menyelesaikan instalasi.

Perhatian: Chrome mungkin menampilkan peringatan bahwa ekstensi dari luar Chrome Web Store tidak aman. Anda dapat mengabaikan peringatan ini jika Anda mempercayai sumber ekstensi.
